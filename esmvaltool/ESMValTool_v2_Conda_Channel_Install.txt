Install via conda channel v2
============================

from scratch (including installer for python and qtm A LOT OF TIME and DISK): conda env create --file environment.yml --name esmvaltool
with linking to existing conda modules (HIGHLY RECOMMENDED): conda env update --file environment.yml --name esmvaltool
source activate esmvaltool
python setup.py install
